pure logical function analytic::usebackward ( real(dp), intent(in)  a,
real(dp), intent(in)  b 
)

Querey if to use forward or backwards propagation.

Only switch to backward when the forward spurious-mode growth (|r1 r2| = B/A) would actually exceed double precision over kL steps. Threshold: forward is considered stable as long as (B/A)^kL < 1e10, i.e. spurious amplitude stays within ~1e10 of the particular solution. Near-degenerate kl (|r1|~|r2|~1) fall in the forward branch, where zero-seed Miller is known to misconverge (spurious modes never damp). Formula: kL * ln(B/A) < ln(1e10) -> B/A < exp(ln(1e10)/kL).

Parameters
[in]a
[in]b
Returns
true if log(a/b) > ln(1E10)
